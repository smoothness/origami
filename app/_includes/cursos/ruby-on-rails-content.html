Objetivo Principal:

Promover en los participantes el desarrollo de capacidades para el uso intermedio del Framework Ruby on Rails para la creación de aplicaciones Web.

Objetivos Específicos:

  • Aprender los conceptos básicos de Ruby como lenguaje de programación.
  • Conocer la estructura de una aplicación de Ruby on Rails y asi entender para qué sirve cada carpeta dentro del proyecto.
  • Desarrollar nuevas funcionalidades a partir del código generado por el framework.
  • Familiarizarse con herramientas para liberar una versión de producción de un proyecto de Ruby on Rails.
  • Conocer sobre algunas Gemas que facilitan el desarrollo de aplicaciones de Ruby on Rails.

Dirigido a:

  • Profesionales de desarrollo Web que trabajan con otras herramientas o lenguajes, pero que desean aprender sobre Ruby on Rails.
  • Desarrolladores front-end que deseen conocer un poco más sobre Ruby on Rails como backend.

Pre requisitos & preparación:

  • Laptop (Sistema operativo preferiblemente basado en Unix, pero se puede trabajar con VM también).
  • Preferiblemente con algunos conocimientos sobre herramientas de consola Unix.
  • Conocimientos de programación (idealmente algún lenguaje de tipo scripting como PHP o Python.
