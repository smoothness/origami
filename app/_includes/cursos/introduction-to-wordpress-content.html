Objetivo Principal:

Ofrecer al estudiante los conocimientos necesarios para desarrollar páginas web de alta calidad utilizando el CMS (Content Management System) WordPress. Repasar las mejores prácticas para poder ofrecer la mayor variedad de posibilidades a clientes con requerimientos básicos y medios.

Dirigido a:

  • Diseñadores y desarrolladores web interesados en incorporar WordPress a sus herramientas o tecnologías de desarrollo.

Pre requisitos & preparación:

  • HTML / CSS
  • JS básico
  • Computadora Portátil (preferiblemente Mac)
