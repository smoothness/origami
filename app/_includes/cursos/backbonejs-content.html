Objetivo Principal:

Hace poco tiempo atrás, las aplicaciones web apostaban al servidor para realizar la mayoría de la manipulación de datos. Hoy en día esto ha cambiado y el servidor envía datos en crudo para ser manipulados por el browser. Como respuesta a esta complejidad que se agregó al lado del cliente, nacieron las librerías MV* (Backbone.js, Angular.js, Ember.js, entre otras).

El objetivo de este curso es formar una base sólida de conocimiento sobre la librería Backbone y utilizar este conocimiento para escribir aplicaciones JavaScript estructuradas y escalables.

Dirigido a:

  • Programadores con interés en aprender a escribir código JavaScript estructurado y escalable mediante la aplicación de patrones de diseño como lo es: MV* (Modelo ­ Vista ­ Presentador / Controlador / etc.)
  • Programadores interesados en profundizar en los detalles de la implementación de “Single Page Applications”

Pre requisitos & preparación:

  • HTML
  • CSS
  • JavaScript itermedio
  • Laptop (Mac o Ubuntu preferiblemente)
