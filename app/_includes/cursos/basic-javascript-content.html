Objetivo Principal:

JavaScript, HTML y CSS se han hecho tan predominantes que ya muchos sistemas operativos han adoptado los estándares web libres como la capa de presentación para aplicaciones nativas, incluyendo Windows 8, Firefox OS, Gnome y Google Chrome OS. También los dispositivos móviles como el iPhone y Android dan soporte a vistas web que permiten incorporar la funcionalidad de JavaScript y HTML5 en aplicaciones nativas. JavaScript también se ha instalado en el lado del servidor. En este curso, probablemente el más importante para cualquier persona que quiera hacer una carrera en desarrollo web, UX, UI developer o cualquier area de desarrollo, aprenderán todos los aspectos principales de este lenguaje tan cotizado en el medio en los últimos tiempos.

Dirigido a:

  • Diseñadores web o desarrolladores de HTML y CSS con deseos de aprender a programar en el lenguaje de programación de la web, JavaScript, para agregarle contenido dinámico a sus proyectos.
  • Desarrolladores de Back-End o de software que deseen aprender el lenguaje de scripting de la web.

Pre requisitos & preparación:

  • HTML
  • CSS
  • Laptop (Mac preferiblemente)
