Objetivo Principal:

Ante el auge y crecimiento de JavaScript del lado cliente (front-end), Node.js permite utilizar el mismo lenguaje del lado servidor (server-side) permitiendo a los desarrolladores crear aplicaciones web de inicio a fin con JavaScript.

Al final de este curso los estudiantes comprenderán los conceptos básicos, herramientas y cómo depurar aplicaciones con Node.js, preparando el camino para un segundo curso más avanzado y profundo.

Dirigido a:

  • Programadores Front-End con interés en ampliar su conocimiento más allá del lado del cliente aprovechando sus conocimientos en JavaScript para desarrollar aplicaciones escalables en tiempo real con Node.js.
  • Programadores Back-End con conocimientos básicos en JavaScript que deseen aplicar sus habilidades del lado de servidor como HTTP y Routing en combinación con Node.js para ampliar sus capacidades en Back-End.

Pre requisitos & preparación:

  • JavaScript básico
  • Comprensión básica de modelo cliente-servidor
  • Computadora Portátil (preferiblemente Mac)
